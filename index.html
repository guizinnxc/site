Tutorial: publicar um site no GitHub Pages

Neste tutorial bem simples, você aprende a colocar seu site estático (HTML/CSS) no ar de forma gratuita.

1. Criar repositório

Entre no GitHub, clique em New repository e escolha um nome para ele.

2. Fazer o upload dos arquivos

Envie seus arquivos: index.html e style.css podem ser arrastados diretamente.

3. Ativar GitHub Pages

Acesse Settings → Pages, selecione a branch main (ou master) e a raiz / (root). Clique em “Save”.

4. Site publicado

Depois de um pouco, o GitHub exibirá o link do seu site:

https://seunome.github.io/nome-do-repositorio

🎉 Parabéns! Seu site agora está no ar.
